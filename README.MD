# DriveGuard

Final year project for the Technical University of Sofia. Represents an automotive vehicle driver safety system based on eye detection with Google Vision API.

## Requirements

### Server

```
JDK 1.8
Spring Boot Framework
MySQL Server
```

### Android Client

```
Android SDK 29
Google Vision API
```
